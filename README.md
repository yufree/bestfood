# 时令果蔬助手 - 微信小程序

一个查询中国各地应季水果蔬菜的微信小程序，帮助用户了解当前时间什么果蔬最好吃、最有营养。

## 🎉 多平台支持

- ✅ **HarmonyOS** - 完整支持
- ✅ **Android** - 完整支持
- ✅ **iOS** - 完整支持
- ✅ **跨平台** - 一套代码运行

## 🌟 功能特点

- **今日推荐**：查看当前季节的高分推荐果蔬
- **搜索查询**：快速搜索特定果蔬
- **智能筛选**：按品类（水果/蔬菜）、来源（国产/进口）筛选
- **季节信息**：显示高峰月份，了解最佳品尝时间
- **营养标签**：查看果蔬的营养价值和功效
- **收藏功能**：收藏喜欢的果蔬

## 📱 页面结构

### 1. 首页 - 今日推荐
- 显示当前日期的应季果蔬推荐
- 分为"今日最佳推荐"和"当季全部果蔬"
- 支持按品类和来源筛选
- 支持日期选择

### 2. 搜索页
- 关键词搜索果蔬
- 实时显示搜索结果
- 热门搜索推荐
- 筛选功能

## 🛠️ 技术栈

- **框架**：微信小程序原生开发
- **数据存储**：本地存储（LocalStorage）
- **日期处理**：原生 JavaScript Date 对象
- **样式**：WXSS（微信小程序样式语言）
- **平台支持**：HarmonyOS / Android / iOS
- **兼容性**：基础库 2.x+ / 3.x+

## 🚀 快速开始

- 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择本项目目录
4. 输入 AppID（测试可以使用测试号）

- 点击"编译"按钮
- 在模拟器中预览效果

2. 填写版本号和项目备注
